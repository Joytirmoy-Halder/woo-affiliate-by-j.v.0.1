# WooCommerce Affiliate System by J

A lightweight and secure affiliate commission tracking plugin for WooCommerce.

## Features
- **Affiliate Commission Tracking**: Automatically track commissions for referrals.
- **Processed Status**: Mark commissions as "Processed" once paid manually.
- **Referral History**: Affiliates can see their referral history (Order ID, Amount, Date, Status) on their dashboard.
- **Admin Search**: Easily search commissions by Order ID or Affiliate Username/Email.
- **Stat Cards**: Dashboard overview with "Approved (Unpaid)" and "Total Paid" stats.
- **Robust Settings**: Manual save handler to prevent critical errors.

## Installation
1. Upload the `woo-affiliate-by-j` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure settings under **WooCommerce > Affiliate System**.

## How to use
- **Admin**: Go to **WooCommerce > Affiliate Commissions** to manage referrals. Use the "Mark Paid" button to set status to `processed`.
- **Affiliates**: View the referral history and earnings on the affiliate dashboard.
